# 小說閱讀器  
  
### 🚀主要功能  
自動抓取正文，提供字體調整、自動捲動等功能，提升小說閱讀體驗。  
  
### 🛠️安裝指南  
1. 瀏覽器需先安裝**篡改猴**元件(TamperMonkey)
2. 安裝**小說閱讀器**腳本 : [main.user.js](https://raw.githubusercontent.com/naimiliu/novelreader/main/main.user.js)
3. **reload** 小說網頁或**重啟**瀏覽器即可  
  
### 🌍支援網址格式  
- https://&lt;domain>/read/&lt;number1>_&lt;number2>.html
- https://&lt;domain>/read/&lt;number1>/p&lt;number2>.html
- https://&lt;domain>/read/&lt;number1>/?&lt;number2>[_&lt;number3>].html
- https://&lt;domain>/&lt;number1>/&lt;number2>[_&lt;number3>]
- https://&lt;domain>/&lt;number1>_&lt;number2>/&lt;number3>.html
- https://&lt;domain>/chapter/&lt;number1>/&lt;number2>.html
- https://&lt;domain>/chapter/&lt;number1>/&lt;number2>/
- https://&lt;domain>/book/&lt;number1>/&lt;number2>.html
- https://&lt;domain>/book_&lt;number1>/&lt;number2>.html
- https://&lt;domain>/Book/Read/&lt;number1>,&lt;number2>
- https://&lt;domain>/txt/&lt;number1>/&lt;number2>
- https://&lt;domain>/txt/&lt;number1>/&lt;number2>.html
- https://&lt;domain>/html/&lt;number1>/&lt;number2>/&lt;number3>.html
